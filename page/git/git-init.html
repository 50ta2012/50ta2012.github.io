開始版本控制

建立一個資料夾，並且初始化這個目錄，讓 Git 對這個目錄開始進行版本控制：

$ mkdir git-practice
$ cd git-practice
$ git init

在資料夾內建立檔案或者修改檔案，完成一個段落之後可以將所有更動過的檔案加入暫存區：

$ git add --all

用指令 git commit 將暫存區的內容放入儲存庫，參數 -m 可以說明這次 commit 的內容：

$ git commit -m "我增加了 ooo, 改了 xxx 呦！"

檢示版本記錄：

$ git log --oneline

[特殊] 追加檔案到最近一次的 Commit：

# 追加檔案到最近一次的 commit
$ git commit --amend --no-edit

# 修改 commit 說明
$ git commit --amend -m "YOUR_NEW_DESCRIPTION"
